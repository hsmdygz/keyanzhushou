3. 图标设计风格建议:
   - 扁平设计风格(Flat Design)或线性图标(Line Icons)
   - 线条粗细统一，建议2px
   - 圆角处理，保持风格一致
   - 图标尺寸建议为48x48像素，PNG透明背景格式

4. 颜色规范:
   - 主色调: 蓝紫色 #5677fc 或 #6c5ce7
   - 功能区域: 翻译功能使用绿色 #00b894，写作功能使用红色 #ff7675
   - 次要导航: 灰色 #999 或 #a0a0a0
   - 彩色背景上的图标: 白色 #FFFFFF
   - 错误提示: 红色 #ff4757

5. 代码修改要点:
   - 需要修改src/static/styles/iconfont.scss中的字体图标引用，改为使用图片
   - 需要将writing.vue、upload.vue、analysis.vue等文件中的字体图标使用更改为图片资源
   - 图标引用方式参考: <image src="/static/icons/icon-name.png" class="icon-size" mode="aspectFit"></image>
   - 建议使用IconImage组件统一管理图标: <icon-image type="icon-name" :size="size"></icon-image>

6. 字体图标迁移示例:

# 原字体图标代码:
```vue
<text class="tool-icon tool-icon-ref">&#xe65c;</text>
```

# 修改为图片资源:
```vue
<image src="/static/icons/optimize.png" class="icon-medium" mode="aspectFit"></image>
```

# 或使用IconImage组件:
```vue
<icon-image type="optimize" :size="24"></icon-image>
```

# 修改示例(writing.vue):
原代码:
```
<view class="tool-btn" @click="insertReference">
  <text class="tool-icon tool-icon-ref">&#xe65c;</text>
  <text class="tool-text">插入引用</text>
</view>
```

修改后:
```
<view class="tool-btn" @click="insertReference">
  <icon-image type="optimize" :size="24"></icon-image>
  <text class="tool-text">插入引用</text>
</view>
```

记得在组件中引入IconImage:
```
import IconImage from '@/components/IconImage.vue';

export default {
  components: {
    PageHeader,
    IconImage
  },
  // ...
}
```

# 需要添加的import语句:
```
import IconImage from '@/components/IconImage.vue';
```
